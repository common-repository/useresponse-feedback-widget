Collect feedback within your WordPress website with an easy-to-use and customizable widget from UseResponse.  

== Description ==
UseResponse Feedback Widget WP plugin is an effective tool to collect feedback from your website for further analysis and decision making. 

The main features of the Feedback Widget:
- Appearance customization (color scheme, on-click unfold type, custom CSS);
- Feedback differentiation (ideas, problems, questions or custom types);
- Multilingual and multi-brand support;
- Knowledge base integration;
- Custom fields;
- Single Sign-on;
- Workflows automatization;
- Analytics and reports.

How to use Feedback Widget to Collect and Process Feedback:
1. Install Feedback Widget on your Wordpress site;
2. Acquire feedback with the help of the widget;
3. Discuss incoming feedback with your colleagues and customers in UseResponse community portal;
4. Analyze and come up with better product decision with the help of built-in UseResponse analytics tool and kanban board. 

Sign up for [free trial](https://www.useresponse.com/trial).

Please note, that UseResponse feedback widget is free to use, but to use it, you need an account at UseResponse. 
To check available plans please visit [Pricing page](https://www.useresponse.com/pricing). There is a 14-day free no obligation [trial](https://www.useresponse.com/trial) to test all the features. 

== Installation ==

1. Install and activate the WordPress Plugin
2. Login to the Admin area in Useresponse. If you don't have UseResponse account, you can sign up for free 14-day trial [here](https://www.useresponse.com/trial).
3. Customize feedback widget from the backend interface of UseResponse and copy the code generated automatically. Instructions can be found [here](https://help.useresponse.com/knowledge-base/article/external-widgets-feedback-support-from-website).
4. Go to UseResponse Feedback Widget plugin in your WordPress admin area.
5. Paste the code and save the settings.


== Frequently Asked Questions ==

= Do I need UseResponse account to use this plugin? =

Yes, UseResponse account is required. You can create it [here](https://www.useresponse.com/trial).

= Do you offer a free trial? =

Yes, We offer free [14-day trial](https://www.useresponse.com/trial) that includes all the features available. No obligations, no credit card required.

= Where can I use Feedback Widget? =

You can place the feedback widget into your website, blog, web application and mobile app.

= What types of feedback do you have? =

By default we have Ideas, Questions, Problems and Thanks. You can disble any types and change the wordings. 

= Can I use SSO? =

UseResponse supports single sign-on using SAML 2.0  with identity provider to be ADFS server. It's provided by Microsoft with ability to use active directory credentials to authenticate in UseResponse.

= Can I use custom domain? =

All our trials within Cloud Plan are created on .useresponse.com domain. But your users want to see your URL in order to make their experience using your support center smooth. So you can point our domain to your own domain.

= Can I create multiple forums? =

Yes, UseResponse allows you to create several forums for different teams in the company, separate products or organize multilingual platforms and manage all these communities in one backend. 

= What languages does UseResponse Feedback widget support? =

We currently support 13 languages, plus any user can easily create their own translation and change the wording of his system. 
So far we have a translation for the following languages: 
- English
- Russian
- French (France)
- Spanish
- German
- Chinese
- Thai
- Dutch
- Portuguese (Brazil)
- Hungarian
- Japanese
- Chinese (Taiwan)
